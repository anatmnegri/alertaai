# Alerta.AI — Painel de Monitoramento

Painel de monitoramento de riscos climáticos e ocorrências urbanas desenvolvido para a Defesa Civil de Recife. Exibe chamados em tempo real, geomapeamento de ocorrências e indicadores de prioridade.


## Tecnologias

- [React 19](https://react.dev)
- [Vite](https://vitejs.dev)
- [Tailwind CSS 4](https://tailwindcss.com)
- [Recharts](https://recharts.org) — gráficos
- [React Leaflet](https://react-leaflet.js.org) + [Leaflet](https://leafletjs.com) — mapa interativo
- [Lucide React](https://lucide.dev) — ícones

## Como rodar

```bash
npm install
npm run dev
```

Acesse em `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```
